# Diary_WhatsUrName-
仿“你的日记”——电影《你的名字》日记app
-----------------------------
在看电影《你的名字》时就有注意到这款app，在App Store上搜了一下，感觉“你的日记”这款应该是最有诚意的。<br>
后续将添加TouchID等功能。<br>
自己仿照这个也写了一个，开源给大家。<br>我的邮箱：wangguanyu@bupt.edu.cn，有问题可以邮件我。<br>
